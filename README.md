# Proiect-Termostat-
Regulator de Temperatura folosind PID

Se foloseste:
- 1 x Arduino
- 1 x StepUp Converter(5 to 12v)
- 1 x Tranzistor de putere
- 4 x Butoane + Rezistente 10k ohm pentru fiecare
- 1 x LCD cu I2C
- 1 x Baterie externa
- 1 x LM35 senzor de temperatura
  
Valori pentru PID:
- P : 10.00
- I : 0.40
- D : 0
